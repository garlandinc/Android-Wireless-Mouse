Mouse Mode

In mouse mode you have lot of option with touch-pad; The Touch-pad contains two block one is big block another is tiny block. The big block is simple touch pad and it move the cursor on your PC and support single click and double click for generate left mouse button and right mouse button click and long click perform the Right mouse button click.
and the Tiny Block is used for scrolling vertically anything like mouse wheel and if you single click on the Tiny block it generate The Mouse middle Click. You can integrate also more things go on top right click menu option and then functionality: and choose what you want with mouse. if you choose mouse button then you can perform Drag and Drop/.

Drag and Drop

Go to Mouse Mode, then make sure Mouse button is appear, if not go to top right option:then Functionality and then checked Mouse Button
Then You see there three Button two is big (left and right mouse button) and one is square check box if you click on check box the left mouse button is pressed and hold, that means it is enabled for drag and drop.
Example: Go to desktop
Move and put your cursor to any Icon or application frame.
Then Click the small Square checkbox button
Then move the cursor with touch-pad to where you want to drop it
Then click again the checkbox button to release the drop
and See what happened..

Finding your PC IP Address

At first you have to connect your PC with any network, like wifi hotspot or router (No need to have Internet) Go to Start menu and Search cmd then you can see Command Prompt and click to run it or press Widows Key and R key together and you see the run box appear and input there cmd and press ok then you can see the Command Prompt Console
On there you type the word inside bracket [ipconfig] and press enter Then you see the [Wireless LAN adapter Wi-Fi] or something else relatively close the sentence. and from there you choose the [IPv4 Address] like here it shows
IPv4 Address ........ 127.68.93.0 or something else and here the part 127.68.93.0 is your PC WIFI IP Address.

Connectivity Problem

The Common problem may occur with you, if your pc shared the static IP address and you have to solve it, by follow the instruction..
Procedure : On the toolbar of your windows PC at the right bottom corner you can see the network option
Right Click on it then Select Open Network and Sharing Center or simply you can go to control panel and search for Network and Sharing Center
Then On the left side you can see the option Change Adapter Settings
Then select your Wifi signal and right click on it and select properties
Then Select Internet Protocol Version 4(TCP/IPv4) from list. and from below there is an option properties click there
Then you there see some option, forgot about them make sure that The Radio Buttons are checked for 1. Obtain IP Address Automatically. 2. Obtain DNS server address Automatically. then press ok. then everything is ok, close all. Then Connect your mouse again.
